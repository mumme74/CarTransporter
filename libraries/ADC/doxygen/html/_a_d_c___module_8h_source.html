38 /* MK20DX256 Datasheet:
39 The 16-bit accuracy specifications listed in Table 24 and Table 25 are achievable on the
40 differential pins ADCx_DP0, ADCx_DM0
41 All other ADC channels meet the 13-bit differential/12-bit single-ended accuracy
42 specifications.
43 
44 The results in this data sheet were derived from a system which has < 8 Ohm analog source resistance. The RAS/CAS
45 time constant should be kept to < 1ns.
46 
47 ADC clock should be 2 to 12 MHz for 16 bit mode
48 ADC clock should be 1 to 18 MHz for 8-12 bit mode
49 To use the maximum ADC conversion clock frequency, the ADHSC bit must be set and the ADLPC bit must be clear
50 
51 The ADHSC bit is used to configure a higher clock input frequency. This will allow
52 faster overall conversion times. To meet internal ADC timing requirements, the ADHSC
53 bit adds additional ADCK cycles. Conversions with ADHSC = 1 take two more ADCK
54 cycles. ADHSC should be used when the ADCLK exceeds the limit for ADHSC = 0.

180 // Settings for the power/speed of conversions/sampling
181 /* For conversion speeds:
182  ADC_VERY_LOW_SPEED is guaranteed to be the lowest possible speed within specs for resolutions less than 16 bits (higher than 1 MHz),
183  it's different from ADC_LOW_SPEED only for 24, 4 or 2 MHz.
184  ADC_LOW_SPEED is guaranteed to be the lowest possible speed within specs for all resolutions (higher than 2 MHz).
185  ADC_MED_SPEED is always >= ADC_LOW_SPEED and <= ADC_HIGH_SPEED.
186  ADC_HIGH_SPEED_16BITS is guaranteed to be the highest possible speed within specs for all resolutions (lower or eq than 12 MHz).
187  ADC_HIGH_SPEED is guaranteed to be the highest possible speed within specs for resolutions less than 16 bits (lower or eq than 18 MHz).
188  ADC_VERY_HIGH_SPEED may be out of specs, it's different from ADC_HIGH_SPEED only for 48, 40 or 24 MHz.
189 */
190 /* For sampling speeds:
191  ADC_VERY_LOW_SPEED is the lowest possible sampling speed (+24 ADCK).
192  ADC_LOW_SPEED adds +16 ADCK.
193  ADC_MED_SPEED adds +10 ADCK.
194  ADC_HIGH_SPEED (or ADC_HIGH_SPEED_16BITS) adds +6 ADCK.
195  ADC_VERY_HIGH_SPEED is the highest possible sampling speed (0 ADCK added).
196 */
197 #define ADC_VERY_LOW_SPEED 0
198 #define ADC_LOW_SPEED 1
199 #define ADC_MED_SPEED 2
200 #define ADC_HIGH_SPEED_16BITS 3
201 #define ADC_HIGH_SPEED 4
202 #define ADC_VERY_HIGH_SPEED 5
